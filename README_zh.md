# Auths by StealthChampions

Auths 是一款高端、现代化的双重验证 (2FA) 代码生成器浏览器扩展。基于 React 和 WXT 框架构建，它在提供卓越视觉体验的同时，确保了极高的数据安全性。

### ✨ 核心特性
- **高端 UI**: 现代化的玻璃拟物化设计，流畅的交互转换。
- **智能品牌识别**: 热门服务平台图标自动匹配展示。
- **账户文件夹**: 支持通过文件夹对账户进行分类，并提供快速筛选标签。
- **实时 TOTP**: 验证码即时生成，配合高保重进度指示器。
- **安全优先**: 使用 Argon2 和 AES 进行本地加密；支持自动锁定。
- **云端同步**: 支持通过 WebDAV, Google Drive 或 Dropbox 进行备份。

### 🚀 快速开始
1. **安装依赖**: `npm install`
2. **启动开发服务器**: `npm run dev`
3. **加载扩展**: 打开 `chrome://extensions/`，开启开发者模式，点击“加载已解解压的扩展程序”，选择 `.output/chrome-mv3` 目录。
