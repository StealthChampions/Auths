**Auths** is a secure and modern 2-Step Verification (2FA / TOTP) browser extension. Built with React and the WXT framework, it provides a clean interface and robust security to manage your authentication codes.

---

## ‚ú® Features

- **Modern UI**: Clean design with dark mode support and smooth transitions.
- **Secure Storage**: Data is encrypted locally using **AES-256** and **Argon2** hashing. Keys never leave your device unencrypted.
- **Organized**: Group accounts into folders with custom icons. The **Smart Filter** shows relevant accounts for the current website.
- **Brand Icons**: Automatically matches icons for popular services.
- **Real-time TOTP**: Accurate code generation with progress indicators.
- **Cloud Sync (Optional)**: Backup and sync your encrypted vault via **WebDAV** (Google Drive and Dropbox support coming soon).
- **QR Code Scanner**: Scan QR codes directly from webpages to add accounts.

## üõ† Tech Stack

- **Framework**: [WXT](https://wxt.dev) (Web Extension Tools)

## üöÄ Quick Start

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/StealthChampions/Auths.git
   cd Auths
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start Development Server**:
   ```bash
   npm run dev
   ```
   This will auto-open a Chrome instance with the extension loaded.

4. **Build for Production**:
   ```bash
   npm run build
   ```
   The output `zip` and folder will be generating in the `.output/` directory.

### Loading Manually

1. Visit `chrome://extensions/` in your browser.
2. Enable **Developer mode** in the top right.
3. Click **Load unpacked**.
4. Select the `.output/chrome-mv3` folder generated by the build process.
