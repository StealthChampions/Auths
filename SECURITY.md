# Security Policy

## Supported Versions

We support the latest versions published on the following extension stores:

- **Chrome Web Store** (Google Chrome)
- **Microsoft Edge Add-ons** (Microsoft Edge)
- **Firefox Add-ons** (Mozilla Firefox)

## Reporting a Vulnerability

Report potential vulnerabilities privately via [this form](https://github.com/StealthChampions/Auths/security/advisories/new).
Where appropriate, include a proof-of-concept and reproduction steps.
We strive to provide an initial response within five days, but as this is a volunteer-run project, we make no guarantees.
